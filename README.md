# 开发

使用Gulp/PostCSS编译样式以满足未来的CSS规范。你需要全局安装 [Node](https://nodejs.org/), [Yarn](https://yarnpkg.com/) 和[Gulp](https://gulpjs.com)。 然后进入主题根目录执行:

```bash
# 安装依赖
yarn

# 运行编译和监控文件
$ yarn dev
```



打包

```bash
yarn zip
```
